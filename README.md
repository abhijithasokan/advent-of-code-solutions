## Setup & Usage

1. **Install Conda environment:**
   ```sh
   conda env create -f environment.yml
   conda activate aoc
   ```

2. **Run a solution:**
   ```sh
   cd <year>
   python <day>.py
   ```
   Replace `<year>` and `<day>` with the appropriate folder and script names.
